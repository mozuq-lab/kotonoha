# iOS ガイド付きアクセス 設定ガイド

## 概要

ガイド付きアクセスは、iPadやiPhoneを1つのアプリに固定し、誤操作によるアプリの切り替えを防ぐ機能です。kotonohaを使用する際に、この機能を有効にすることで、安心してコミュニケーションに集中できます。

## 対象デバイス

- iPad（iOS 14.0以上）
- iPhone（iOS 14.0以上）

## 設定手順

### 1. ガイド付きアクセスを有効にする

1. **「設定」アプリ**を開きます
2. **「アクセシビリティ」**をタップします
3. 下にスクロールして**「ガイド付きアクセス」**をタップします
4. **「ガイド付きアクセス」**のスイッチをオンにします

### 2. パスコードを設定する（推奨）

ガイド付きアクセスを終了する際に必要なパスコードを設定します。

1. 「ガイド付きアクセス」の設定画面で**「パスコード設定」**をタップします
2. **「ガイド付きアクセス用のパスコードを設定」**をタップします
3. 4桁のパスコードを入力します（忘れないようにメモしてください）
4. 確認のため、もう一度同じパスコードを入力します

**注意**: パスコードを忘れると、デバイスを再起動するまでガイド付きアクセスを終了できなくなります。

### 3. Face ID/Touch IDを有効にする（オプション）

パスコードの代わりに、Face IDまたはTouch IDでガイド付きアクセスを終了できるようにします。

1. 「パスコード設定」画面で**「Face ID」**または**「Touch ID」**のスイッチをオンにします

## 使用方法

### ガイド付きアクセスを開始する

1. **kotonohaアプリ**を起動します
2. **サイドボタン（電源ボタン）を3回**すばやく押します
3. 「ガイド付きアクセス」の画面が表示されます
4. 画面右上の**「開始」**をタップします

### ガイド付きアクセス中の操作

- アプリは通常どおり使用できます
- ホームボタンやジェスチャーでアプリを閉じることはできません
- 通知は表示されません（設定による）

### ガイド付きアクセスを終了する

1. **サイドボタン（電源ボタン）を3回**すばやく押します
2. **パスコード**を入力します（またはFace ID/Touch IDを使用）
3. 画面左上の**「終了」**をタップします

## 詳細設定

### 無効にする領域を設定する

ガイド付きアクセス開始時に、画面の一部をタップできないように設定できます。

1. サイドボタンを3回押してガイド付きアクセス画面を表示
2. 無効にしたい領域を指で囲みます
3. 「開始」をタップします

### ハードウェアボタンの制限

1. ガイド付きアクセス画面で**「オプション」**をタップします
2. 以下の項目を設定できます：
   - **サイドボタン**: スリープボタンの無効化
   - **音量ボタン**: 音量調整の無効化
   - **動作**: 画面回転などの無効化
   - **ソフトウェアキーボード**: キーボードの無効化
   - **タッチ**: タッチ操作の無効化
   - **時間制限**: 使用時間の制限

## トラブルシューティング

### ガイド付きアクセスが開始できない

- 設定アプリで「ガイド付きアクセス」が有効になっているか確認してください
- サイドボタンを3回押す操作が正しく認識されているか確認してください

### パスコードを忘れた

1. デバイスを強制再起動します
   - Face IDモデル: 音量上ボタンを押してすぐ離す → 音量下ボタンを押してすぐ離す → サイドボタンを長押し
   - ホームボタンモデル: ホームボタンとサイドボタンを同時に長押し
2. 再起動後、設定アプリからパスコードを再設定してください

### ガイド付きアクセスが勝手に終了する

- バッテリー残量が少なくなると、ガイド付きアクセスが終了することがあります
- デバイスを充電してから再度お試しください

## 推奨設定

kotonohaを使用する際の推奨設定：

| 設定項目 | 推奨値 | 理由 |
|---------|--------|------|
| サイドボタン | オフ | 誤ってスリープしないように |
| 音量ボタン | オン | 読み上げ音量を調整できるように |
| タッチ | オン | アプリを操作できるように |
| 動作 | オフ | 画面が回転しないように（対面表示モード使用時を除く） |
