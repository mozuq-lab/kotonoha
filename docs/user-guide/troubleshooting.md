# トラブルシューティングガイド

## 音声読み上げに関する問題

### 読み上げボタンを押しても音が出ない

**考えられる原因と対処法:**

1. **音量がゼロになっている**
   - デバイスの音量ボタンで音量を上げてください
   - iOS: コントロールセンターで音量を確認
   - Android: 設定 → 音 → メディアの音量を確認

2. **マナーモード（サイレントモード）が有効**
   - iOS: 本体側面のスイッチをオフに（オレンジ色が見えない状態）
   - Android: 音量ボタンを押して「通常モード」に切り替え

3. **Bluetoothデバイスに接続されている**
   - Bluetoothイヤホンやスピーカーに接続されていないか確認
   - 接続されている場合は、そのデバイスから音が出ているか確認

4. **TTS（音声合成）が正しく設定されていない**
   - iOS: 設定 → アクセシビリティ → 読み上げコンテンツ → 声 で日本語の声が設定されているか確認
   - Android: 設定 → ユーザー補助 → テキスト読み上げの設定 で日本語が選択されているか確認

### 読み上げの声が変

**対処法:**

1. デバイスの音声設定を確認してください
   - iOS: 設定 → アクセシビリティ → 読み上げコンテンツ → 声
   - Android: 設定 → ユーザー補助 → テキスト読み上げの設定

2. 別の音声エンジンを試してみてください
   - Android: Google Text-to-Speech を使用することを推奨

3. アプリを再起動してみてください

### 読み上げが途中で止まる

**対処法:**

1. 長いテキストの場合は、段落ごとに分けて読み上げることを検討してください
2. デバイスを再起動してみてください
3. アプリのキャッシュをクリアしてみてください

## 文字入力に関する問題

### 文字盤のボタンが反応しない

**対処法:**

1. 画面を強くタップしすぎていないか確認（軽くタップしてください）
2. 画面が汚れている場合は、柔らかい布で拭いてください
3. 画面保護フィルムが原因の場合があります。フィルムを外して試してください
4. アプリを再起動してみてください

### 入力した文字が表示されない

**対処法:**

1. 入力欄がスクロールして見えなくなっている場合があります。上にスクロールしてください
2. アプリを再起動してみてください
3. デバイスを再起動してみてください

### 濁点・半濁点が付かない

**対処法:**

1. 濁点（゛）や半濁点（゜）が付けられる文字は限られています
   - 濁点: か行、さ行、た行、は行
   - 半濁点: は行のみ
2. まず文字を入力してから、濁点・半濁点ボタンをタップしてください

## 定型文に関する問題

### 定型文が保存されない

**対処法:**

1. 定型文の文字数が500文字を超えていないか確認してください
2. アプリを再起動してから再度試してください
3. デバイスのストレージ容量が不足している可能性があります

### 定型文が消えてしまった

**考えられる原因:**

1. アプリをアンインストールすると、すべてのデータが削除されます
2. デバイスの「データを消去」操作をすると削除されます

**予防策:**
- 大切な定型文はメモアプリなどにバックアップしておくことをお勧めします

## AI変換に関する問題

### AI変換ボタンが押せない（グレーアウトしている）

**原因:**
- インターネット接続がない状態です

**対処法:**
1. Wi-Fiまたはモバイルデータ通信に接続してください
2. 機内モードがオフになっているか確認してください
3. 接続後、数秒待ってから再度試してください

### AI変換の結果がおかしい

**対処法:**

1. 「再生成」ボタンをタップして別の候補を取得してください
2. 入力テキストを具体的にすると、より適切な変換結果が得られます
   - 例: 「水」→「お水が欲しい」のように補足すると良い結果に
3. 「元の文」ボタンで変換前のテキストを使用することもできます

### AI変換が遅い

**対処法:**

1. インターネット接続が安定しているか確認してください
2. 混雑時は応答が遅くなる場合があります。しばらく待ってから再度お試しください
3. 3秒以上かかる場合はローディング表示が出ます

### AI変換でエラーが発生する

**対処法:**

1. インターネット接続を確認してください
2. 時間をおいて再度試してください
3. エラーが続く場合は「元の文」を使用してください

## 緊急ボタンに関する問題

### 緊急ボタンを押しても音が出ない

**対処法:**

1. デバイスの音量を最大にしてください
2. マナーモード（サイレントモード）を解除してください
3. 一部のデバイスでは、マナーモード中でも緊急音を鳴らすことができない場合があります

### 緊急アラート画面から戻れない

**対処法:**

1. 画面下部の「リセット」ボタンをタップしてください
2. ボタンが見つからない場合は、画面をスクロールしてください
3. それでも戻れない場合は、アプリを強制終了してください
   - iOS: 下から上にスワイプしてアプリを上にフリック
   - Android: 最近使ったアプリボタン → アプリを上にスワイプ

## 表示に関する問題

### 画面が小さくて見づらい

**対処法:**

1. 設定画面でフォントサイズを「大」に変更してください
2. デバイスの表示設定で文字サイズを大きくしてください
3. 9.7インチ以上のタブレットでの使用を推奨しています

### 画面の色が見づらい

**対処法:**

1. 設定画面でテーマを変更してください
   - 「ライトモード」: 明るい背景
   - 「ダークモード」: 暗い背景
   - 「高コントラストモード」: コントラストを強調
2. デバイスの明るさ設定を調整してください

### 対面表示モードが正しく動作しない

**対処法:**

1. 画面の自動回転が有効になっているか確認してください
2. デバイスを水平に保って試してください
3. アプリを再起動してみてください

## アプリの動作に関する問題

### アプリが起動しない

**対処法:**

1. デバイスを再起動してください
2. アプリを最新バージョンにアップデートしてください
3. アプリを削除して再インストールしてください（データは失われます）

### アプリがフリーズする

**対処法:**

1. アプリを強制終了してください
   - iOS: 下から上にスワイプしてアプリを上にフリック
   - Android: 最近使ったアプリボタン → アプリを上にスワイプ
2. デバイスを再起動してください
3. デバイスのストレージ容量を確認してください（空き容量が少ないと動作が不安定になります）

### アプリが突然終了する

**対処法:**

1. デバイスを再起動してください
2. アプリを最新バージョンにアップデートしてください
3. 他のアプリを終了してメモリを解放してください
4. OSを最新バージョンにアップデートしてください

### 設定が保存されない

**対処法:**

1. 設定変更後、アプリを完全に終了してから再起動してください
2. デバイスのストレージ容量を確認してください
3. アプリを再インストールしてください（データは失われます）

## データに関する問題

### 履歴・定型文が消えた

**考えられる原因:**

1. アプリをアンインストールした
2. デバイスの「アプリのデータを消去」操作をした
3. デバイスを初期化した

**予防策:**
- 大切なデータは定期的にメモアプリなどにバックアップしてください

### データをバックアップしたい

**現在のバージョンでは:**
- 自動バックアップ機能やエクスポート機能は提供されていません
- 大切な定型文やフレーズは、メモアプリなどに手動でコピーしておくことをお勧めします

---

## 接続に関する問題

### オフライン表示が消えない

**対処法:**

1. Wi-Fiまたはモバイルデータ通信が有効か確認してください
2. 機内モードがオフになっているか確認してください
3. 別のWi-Fiネットワークに接続してみてください
4. デバイスを再起動してください

---

## 問題が解決しない場合

上記の対処法を試しても問題が解決しない場合は：

1. **アプリのバージョンを確認**: 設定画面で現在のバージョンを確認し、最新版にアップデートしてください

2. **デバイス情報を確認**:
   - デバイスの機種名
   - OSのバージョン
   - アプリのバージョン

3. **サポートへのお問い合わせ**:
   - アプリ内の「お問い合わせ」からご連絡ください
   - 問題の詳細、発生状況、上記のデバイス情報をお伝えいただくと、より迅速に対応できます
