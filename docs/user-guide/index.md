# kotonoha ユーザーガイド

## 概要

**kotonoha（ことのは）** は、発話が困難な方のためのコミュニケーション支援アプリです。

タブレットやスマートフォンで簡単に文字を入力し、音声で読み上げることで、家族や介護者、医療スタッフとのコミュニケーションをサポートします。

## 主な機能

### 文字入力

- **五十音文字盤**: タップで簡単に文字を入力
- **定型文**: よく使うフレーズをワンタップで入力
- **履歴**: 過去に入力した文章を再利用

### 音声読み上げ

- **TTS読み上げ**: 入力した文章をOS標準の音声合成で読み上げ
- **対面表示モード**: 画面を180度回転させて相手に見せる

### コミュニケーション補助

- **大ボタン**: 「はい」「いいえ」「わからない」をワンタップで
- **状態ボタン**: 「痛い」「トイレ」などの状態を即座に伝える
- **緊急ボタン**: 緊急時に周囲の人に助けを求める

### カスタマイズ

- **フォントサイズ**: 小・中・大の3段階
- **テーマ**: ライト・ダーク・高コントラストの3種類
- **読み上げ速度**: 遅い・普通・速いの3段階

### AI変換（オンライン機能）

- 短いキーワードを自然で丁寧な文章に変換
- カジュアル・普通・丁寧の3段階で変換

## 対応環境

| プラットフォーム | 要件 |
|-----------------|------|
| iOS | iOS 14.0以上（iPad推奨） |
| Android | Android 10以上（タブレット推奨） |
| Web | Chrome、Safari、Edge最新版 |

**推奨**: 9.7インチ以上のタブレットでの使用を推奨しています。

## オフラインでできること

インターネット接続がなくても、以下の機能は利用できます：

- 文字盤入力
- 定型文の選択・使用
- 音声読み上げ
- 大ボタン・状態ボタン
- 緊急ボタン
- 履歴・お気に入りの閲覧
- 設定の変更

**注意**: AI変換機能のみ、インターネット接続が必要です。

## プライバシーについて

- **ローカル保存**: 定型文、履歴、設定はすべて端末内にのみ保存されます
- **AI変換**: AI変換使用時のみ、入力テキストが一時的にサーバーに送信されます（保存されません）
- **完全削除**: アプリをアンインストールすると、すべてのデータが削除されます
