# Android 画面ピン留め 設定ガイド

## 概要

画面ピン留め（Screen Pinning）は、Androidデバイスを1つのアプリに固定し、誤操作によるアプリの切り替えを防ぐ機能です。kotonohaを使用する際に、この機能を有効にすることで、安心してコミュニケーションに集中できます。

## 対象デバイス

- Android 10以上を搭載したスマートフォン・タブレット

**注意**: メーカーによって設定項目の名称や場所が異なる場合があります。

## 設定手順

### 1. 画面ピン留めを有効にする

#### 標準的なAndroid（Pixel、Xperiaなど）

1. **「設定」アプリ**を開きます
2. **「セキュリティ」**または**「セキュリティとプライバシー」**をタップします
3. **「画面ピン留め」**または**「アプリ固定」**をタップします
4. **「画面ピン留めを使用」**のスイッチをオンにします

#### Samsung Galaxy

1. **「設定」アプリ**を開きます
2. **「生体認証とセキュリティ」**をタップします
3. **「その他のセキュリティ設定」**をタップします
4. **「アプリを固定」**のスイッチをオンにします

#### OPPO / Realme

1. **「設定」アプリ**を開きます
2. **「パスワードとセキュリティ」**をタップします
3. **「システムセキュリティ」**をタップします
4. **「アプリロック」**から**「画面固定」**を有効にします

### 2. 解除時のセキュリティを設定する（推奨）

画面ピン留めを解除する際に、PIN、パターン、または生体認証を要求するように設定できます。

1. 画面ピン留めの設定画面で
2. **「解除前にロック解除パターンの入力を求める」**または類似の項目をオンにします

## 使用方法

### 画面ピン留めを開始する

1. **kotonohaアプリ**を起動します
2. **「最近使ったアプリ」ボタン**をタップします（画面下部の□ボタン、またはスワイプジェスチャー）
3. kotonohaのアプリカードが表示されます
4. アプリカード上部の**アプリアイコン**をタップします
5. メニューから**「固定」**または**「ピン留め」**をタップします
6. 確認ダイアログで**「OK」**または**「わかりました」**をタップします

### 画面ピン留め中の操作

- アプリは通常どおり使用できます
- ホームボタンや戻るボタンでアプリを閉じることはできません
- 通知バーへのアクセスが制限されます

### 画面ピン留めを解除する

#### ジェスチャーナビゲーションの場合

1. 画面下部から上にスワイプして**長押し**します
2. セキュリティ設定が有効な場合は、PIN、パターン、または生体認証で解除します

#### ボタンナビゲーションの場合

1. **「戻る」ボタン**と**「最近使ったアプリ」ボタン**を**同時に長押し**します
2. セキュリティ設定が有効な場合は、PIN、パターン、または生体認証で解除します

## メーカー別の詳細設定

### Samsung Galaxy

Samsung独自の「Kids Mode」や「Secure Folder」と組み合わせることで、より強固な保護が可能です。

1. **「設定」**→**「便利な機能」**→**「デジタルウェルビーイング」**
2. 「ペアレンタルコントロール」からアプリの使用制限が可能

### Xiaomi / POCO

1. **「設定」**→**「アプリ」**→**「アプリロック」**
2. kotonohaをロック対象に追加
3. 他のアプリへのアクセスを制限

### OPPO / Realme

1. **「設定」**→**「プライバシー」**→**「キッズスペース」**
2. kotonohaのみを許可アプリに設定

## トラブルシューティング

### 画面ピン留めが見つからない

- 「設定」アプリ内で**「固定」「ピン留め」「アプリロック」**などのキーワードで検索してください
- メーカーによって名称が異なります：
  - Google/Pixel: 「画面ピン留め」
  - Samsung: 「アプリを固定」
  - Xiaomi: 「画面固定」
  - OPPO: 「アプリ固定」

### 最近使ったアプリにピン留めオプションが表示されない

1. 設定で画面ピン留めが有効になっているか確認してください
2. デバイスを再起動してみてください
3. アプリを完全に終了してから再度開いてください

### 解除できない

- **「戻る」**と**「最近使ったアプリ」**ボタンを**同時に**長押ししてください（約3秒間）
- ジェスチャーナビゲーションの場合は、下から上にスワイプして長押しです
- それでも解除できない場合は、デバイスを再起動してください

### 画面ピン留め中に通知が表示される

一部のデバイスでは、画面ピン留め中も通知が表示されます。完全に通知を遮断したい場合：

1. **「設定」**→**「通知」**→**「おやすみモード」**をオンにします
2. または、kotonohaアプリ以外の通知を個別にオフにします

## 推奨設定

kotonohaを使用する際の推奨設定：

| 設定項目 | 推奨値 | 理由 |
|---------|--------|------|
| 画面ピン留め | オン | アプリを固定できるように |
| 解除時の認証 | オン | 誤って解除されないように |
| 音量ボタン | 有効のまま | 読み上げ音量を調整できるように |
| おやすみモード | オプション | 通知に邪魔されたくない場合 |
